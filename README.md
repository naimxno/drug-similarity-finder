# Drug Similarity Finder

## What this project does
Calculates Tanimoto similarity between 20 common drugs 
using RDKit Morgan Fingerprints and visualizes results as a heatmap.

## Tools used
- Python
- RDKit
- Pandas
- Matplotlib
- Seaborn

## Key finding
Serotonin & Melatonin showed highest similarity (0.388) 
due to shared indole ring structure.

## How to run
Open in Google Colab and run all cells.
